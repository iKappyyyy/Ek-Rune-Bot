
# Ek Rune Discord Bot

This is a discord bot designed for creating organized player lobbies for the Guild Raiders discord server!


## Features

- Creating lobbies
- Reserve spots for other users
- Ban users from joining your lobby


## Tech Stack

- discord.js
- DJS Commander
- MongoDB
